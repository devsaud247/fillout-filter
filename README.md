## Fillout FIlter API

### Installation

`npm install`

After installing node_modules you can start the project using below command

### Start Command

`npm start`

### Environment Variables

Make sure to add appropriate values to these environment variables in **.env.local** files

```
// LOCAL CONFIGS
PORT = 8080

// FILLOUT CONFIGS
FILLOUT_API_KEY =
FILLOUT_BASE_URL =

```
